#JPalindromes

Petit exercice en JavaScript permettant de détecter si un mot ou une chaîne de caractères est un palindrome.
Un palindrome est une suite de caractères qui se lit de la même manière dans les deux sens.
Exemples : radar, kayak, level, racecar

##Objectifs pédagogiques

Cet exercice m’a permis de pratiquer :
- La manipulation de chaînes de caractères
- L’inversion de texte
- La création de fonctions réutilisables
- La gestion des cas particuliers (majuscules, espaces, caractères spéciaux)

Langage : JavaScript
